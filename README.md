## Prerequisites
1. `node > v15.x`
2. `yarn > v2.x`

## Usage
1. `yarn && yarn dev`

## Scripts
| yarn | desc |
| ---- | ---- |
| yarn | install dependencies |
| yarn dev | dev server |
| yarn build && yarn start | prod server |
| yarn svgr | convert svg icons to react components |
| yarn data:update | generate / update data from tokens |
| yarn tsc | type check |
| yarn lint | linter |
